ESP8266 智能宠物照护系统
这是一个基于ESP8266开发板制作的只能宠物照护系统


这个系统的初衷是让家里的宠物生活得更舒适。它能根据光线自动开关灯，检测到宠物活动时会自动喂食，还能通过手机远程看到家里传感器的数据 。


具体功能如下：
1. 智能补光：
傍晚/阴天：光线变暗时自动补光 。
白天/深夜：白天光线足或深夜睡觉时无需补光，不打扰宠物休息 。
2. 自动喂食：
检测到宠物活动且距离上次喂食超过一定时间（演示设为 30 秒）时，舵机模拟喂食机会自动打开闸门投喂食物 。
3. 声音感应：
如果你对着它拍拍手或喊一声，灯会自动亮 5 秒钟（类似楼道感应灯）。
4. 手动触摸：
设备上有一个触摸开关，摸一下开灯，再摸一下关灯 。
5. 手机监控：
通过 WiFi 连接到云平台，可以在手机端pc端看到当前的光照、声音和宠物活动状态 。

使用到的硬件：

ESP8266 (NodeMCU) 

BH1750 光照传感器（测亮度）

人体检测传感器（测宠物活动）

高强度麦克风传感器（测声音）

触摸传感器（手动开关）

舵机 （模拟喂食口的闸门）

快速开始
1. 准备环境：在 Arduino IDE 中安装 'BH1750'、'PubSubClient' 和 'Servo' 库 。
2. 修改配置：
在代码里把 'ssid' 和 'password' 改成你家里的 WiFi 名和密码 。
如果使用巴法云，记下你的 'clientId' 。
3. 上传代码：用 USB 线把 ESP8266 连上电脑，点“上传”即可 。
PS:
喂食时间：代码里为了方便演示，设置了每 30 秒可以喂一次。
